TextLens 🔍

Open-source AI Content & Similarity Detection Platform

TextLens is a free, self-hostable alternative to expensive AI-content detection and plagiarism-checking services. It helps students, educators, researchers, and writers analyze text using transparent, open-source methods.

---

✨ Features

AI Content Detection

Estimate how likely a piece of text was generated by an AI model using:

- Perplexity analysis
- Burstiness scoring
- Sentence-level statistical evaluation
- Explainable scoring instead of black-box predictions

Similarity Detection

Compare two pieces of text and identify:

- Semantic similarity
- Near-duplicate content
- Overlapping phrases
- Potential plagiarism indicators

Privacy First

- No user tracking
- No paid subscriptions
- Fully self-hostable
- Open-source codebase

⚙️ How AI Detection Works

TextLens uses statistical signals commonly discussed in AI-text research:

Perplexity

Measures how predictable a text appears to a language model.

- Lower perplexity → more predictable text
- Higher perplexity → more varied text

Burstiness

Measures variation between sentences.

- Human writing often contains greater variation
- AI-generated writing tends to be more uniform

The final score combines these signals to estimate AI-likelihood.

«Important: No AI detector can reliably determine authorship with complete accuracy. Results should be treated as indicators rather than proof.»

---

🔎 Similarity Detection

TextLens compares documents using:

1. TF-IDF vectorization
2. Cosine similarity scoring
3. N-gram overlap analysis

The system highlights matching segments and reports an overall similarity percentage.

📡 API Example

AI Detection

POST /detect-ai

Request

{
  "text": "Your content here"
}

Response

{
  "ai_score": 82,
  "perplexity": 27.5,
  "burstiness": 4.2
}

Similarity Check

POST /similarity

Request

{
  "text_a": "First document",
  "text_b": "Second document"
}

Response

{
  "similarity_score": 76,
  "matches": [
    "matching phrase"
  ]
}
